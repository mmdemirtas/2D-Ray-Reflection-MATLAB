This MATLAB project simulates basic 2D ray tracing with reflection.
A light ray is emitted from a defined origin at a user-specified incident angle and is tested for intersection with a finite line segment representing a mirror.
If an intersection occurs within the mirror boundaries, the reflection direction is computed using the law of reflection in vector form.
The simulation visualizes the incident ray, mirror surface, and reflected ray in a 2D plot, providing a simple geometric demonstration of optical reflection principles.
